# SolidJS Reference (Concise, Opinionated)

Last checked: 2026-03-01 (America/Los_Angeles)

This set is a short decision guide for browser apps (desktop and mobile) using SolidJS with a TypeScript-first bias.

## Read this first
- Keep components simple and local first.
- Add Router only when URL state must be shareable.
- Add SolidStart only when you need server capabilities, SSR, or full-stack routing.
- Add Solid Meta when SEO/share tags and structured head management matter.
